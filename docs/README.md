# Documentation & Templates

Document templates, generated files, and sample data for the Self-Appraisal system.

## Directory Structure

### `/templates`
Master templates for appraisal forms:
- `template.docx` - Base template for generating appraisals
- `MSP Self-Appraisal form.docx` - MSP self-appraisal form template
- `Faculty Appraisal- Corrective Action Report.docx` - Corrective action report template

### `/generated`
Generated and filled documents:
- `filled_template.docx` - Filled appraisal document
- `appfilled_template.docx` - App-generated filled template
- `debug_filled_template.docx` - Debug version of filled template
- `filled_template.pdf` - PDF export of filled template

### `/data`
Sample data and marks files:
- `marks.xlsx` - Marks/ratings spreadsheet
- `marks (1).xlsx` - Backup marks file
